#Getting Started
1. Install Meteor.js `$ curl https://install.meteor.com | /bin/sh`

2. Clone this repository `$ git clone https://github.com/TheABrad/myplaylistapp.git`

3. Navigate to the musicapp directory and run `$ meteor`
